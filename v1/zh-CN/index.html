介绍

什么是 Vonic ？

Vonic 是一个基于 Vue.js 和 Ionic CSS 样式的 UI 框架，用于快速构建移动端单页应用。

Vonic 和 ionic 的关系

没有关系，只是用了ionic框架的样式文件（做了一些细微的调整）而已。
